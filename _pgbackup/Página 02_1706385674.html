CEYIM - Unidades de medida

La Física se encarga del estudio del universo a través del estudio de un sistema físico. Son ejemplos de sistemas físicos una estrella, un haz luminoso, un átomo de un elemento, un resorte, el sistema Tierra-Luna o un circuito eléctrico, entre otros.

El estudio se lleva a cabo a través de la observación y la medición de propiedades o características de un sistema físico.

Magnitudes físicas
Se llama magnitud a todo aquello que puede ser medido.

Las propiedades que son susceptibles de ser medidas, reciben el nombre de magnitudes físicas. Así, la longitud, la masa, la velocidad, el tiempo y la temperatura, son ejemplos de magnitudes físicas.

Otras propiedades, como el olor, el sabor, la bondad, la belleza, no son magnitudes físicas, ya que no se pueden medir.

Existen magnitudes físicas que son independientes de las demás y reciben el nombre de magnitudes fundamentales; son aquellas que no se definen en función de otras magnitudes físicas, entre ellas mencionamos la longitud, la masa y el tiempo.

Algunas magnitudes se definen a partir de las magnitudes fundamentales y reciben el nombre de magnitudes derivadas. Las magnitudes derivadas resultan de multiplicar o dividir entre sí las magnitudes fundamentales. Por ejemplo, la velocidad de un objeto se obtiene a partir de la división de la longitud entre el tiempo, por lo tanto, la velocidad es una magnitud derivada.

Unidad de medida
Es el valor de una propiedad (generalmente es igual a 1) de un objeto que se toma como referencia (unidad de medida o patrón). Una de las características principales que debe cumplir un patrón de medida es que sea reproducible e invariable.

Medir
Consiste en determinar la magnitud (propiedad) de un cuerpo por comparación con la magnitud de otro cuerpo que se toma como referencia (unidad o patrón de medida).

Para la descripción de un sistema físico es imprescindible la medición, ya que permite establecer relaciones cuantitativas entre las diversas variables que intervienen.

Las magnitudes físicas se miden utilizando instrumentos calibrados; así, la masa de un cuerpo se puede medir en una balanza de platillos, comparándola con la de otros cuerpos de masa conocida).

El resultado de la medición de una magnitud se expresa mediante un número y una unidad. Por ejemplo, si se mide la altura (l) de una persona y se toma como unidad el metro (m), el resultado debe expresarse de esta manera: l = 1.80 m, donde el número 1.80 indica cuántas unidades (metros en este caso) están contenidas en la magnitud medida (la altura de la persona). Decir únicamente que la altura de la persona es 1.80 no tendría significado, ya que podría tratarse de 1.80 centímetros, 1.80 milímetros, etc.

Sistema de unidades
Es el conjunto de magnitudes y sus respectivas unidades de medición. Existen varios sistemas de unidades.

El sistema cegesimal. Que esta en desuso.

El sistema Ingles. Usado principalmente en los países de habla inglesa.

El Sistema Internacional. De uso generalizado en todos los países para aplicaciones de Ingeniería y ciencias. Este sistema es el que usamos en México.
